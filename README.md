# 🌱 Agro Forte - Futuro Sustentável

Projeto desenvolvido para demonstrar a importância da agricultura sustentável e da preservação ambiental.

## Funcionalidades

- Design responsivo
- Navegação simples
- Cards informativos
- Calculadora Verde
- Interatividade com JavaScript

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Objetivo

Mostrar como o agronegócio pode crescer de forma sustentável, equilibrando produção e preservação do meio ambiente.
